# Interrupting

## Usage

1. Checkout repository.
2. `make`

## Development

1. Checkout repository.
2. Verify go is installed.

```sh
brew install go
```

3. Verify [`pipenv` is installed](https://pipenv.pypa.io/en/latest/installation.html).

```sh
pip install pipenv --user # installs pipenv to your user's home directory
```

4. `make dev`
